# Scribejam — Future Extensions

Post-MVP features to build after the Tier 1 core loop is solid.

## Tier 2 — Power Features

| Feature | Description |
|---------|-------------|
| **Templates** | Customizable note structures (1:1, standup, sales call, interview). Users set a template before or after a meeting to shape AI output. |
| **Contextual chat (Cmd+J)** | AI assistant that knows the full meeting context. Ask mid-meeting questions like "What were the action items?" with source citations from the transcript. |
| **Recipes** | Saved AI prompts that combine instructions with real meeting context, invoked via "/" in the chat. Examples: leadership coaching, sales call analysis, customer discovery synthesis. |
| **Note history** | Full searchable archive of past meetings with full-text search across notes and transcripts. |
| **Sharing** | Generate public links for sharing enhanced notes. Recipients can view without a Scribejam account. |

## Tier 3 — Team & Integration

| Feature | Description |
|---------|-------------|
| **Shared folders** | Team spaces (Sales Calls, Customer Feedback, Hiring Loops) that anyone on the team can access. |
| **Cross-meeting analysis** | Batch queries across folders — e.g., "What are the most common feature requests from Q1 customer calls?" |
| **Slack integration** | Post concise meeting summaries to relevant Slack channels. |
| **CRM integrations** | Match meeting notes to CRM records, push summaries to HubSpot, Attio, Affinity. |
| **Notion integration** | Export summaries and push notes into Notion knowledge base. |
| **Zapier/webhook** | Generic integration layer for connecting to thousands of apps. |
